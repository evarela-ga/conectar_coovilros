# conectar_coovilros

Informe tecnico-ejecutivo de relevamiento de necesidades para la Cooperativa de Servicios Publicos de Villa del Rosario. Elaborado por GoodApps.

## Sitio web

https://evarela-ga.github.io/conectar_coovilros/

## Contenido

Archivo index.html: informe completo de 15 secciones (resumen ejecutivo, relevamiento de necesidades, diagnostico tecnico, soluciones propuestas, requerimientos, estimacion de esfuerzo, infraestructura, plan de implementacion, soporte, riesgos y anexos).

## Necesidades identificadas

Necesidad 1 - Plataforma de integracion y analitica de medidores domiciliarios, sobre los datos del MDM Connector en SQL Server.

Necesidad 2 - Adquisicion y normalizacion de datos de medidores de Media Tension, con protocolos heterogeneos y en parte propietarios.

Documento de uso interno. Version v1.0 - 23/06/2026.
